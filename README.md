## Hackathon Quick Start

1. **Pick a simple idea** — you can build more than you think
2. **Write a PRD** — even just 5 bullet points
3. **Set up CLAUDE.md** — give the agent your project context
4. **Let the agent plan first** — review the plan before implementing
5. **Work in small increments** — one feature at a time
